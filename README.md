# Projeto de Detecção de Fraudes em Imagens Automotivas


AutoShield-AI: Projeto de Detecção de Fraudes em Imagens Automotivas

Algumas semanas desenvolvi o AutoShield-AI, um sistema conceitual de detecção de fraudes em imagens automotivas usando análise forense digital.

Com o avanço das IAs generativas, criei um projeto que simula como seria um sistema para identificar se imagens de acidentes/reparos veiculares foram geradas artificialmente - um problema hipotético mas plausível no mercado de seguros.

O que funciona:
- Interface web com upload de imagens
- Análise heurística de padrões digitais
- Verificação de metadados EXIF
- Geração de relatórios em PDF

Ferramentas: Python, FastAPI, Pillow, ReportLab.

Este é um projeto de estudos/portfólio. O sistema usa análise heurística básica e não foi validado em cenários reais de detecção de fraudes.

Poderia integrar modelos de deep learning reais, APIs corporativas, banco de dados robusto e dashboard analytics.

Um projeto interessante que me ensinou sobre arquitetura de APIs, resolução de problemas técnicos e visão de produto.
